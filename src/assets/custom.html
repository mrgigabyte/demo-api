How to begin?

The API has 3 user roles namely:

  • GOD : With the highest privileges
  • JESUS : With less privileges than GOD
  • ROMANS : With the least privileges

Do a request to /user/login endpoint. This endpoint will return 3 jwts for each type of user privileges. Copy/paste the value of any one of these JWTS in the page header where it is written api_key so that you can access routes that require authentication.

Generic Error Codes

  • 400 Bad Request : The request could not be understood by the server due to malformed syntax.
  • 401 Unauthorised: Missing/Wrong authentication information in the request.
  • 404 Not Found: The server has not found anything matching the request-uri.
  • 500 Internal Server Error: The request could not be fulfilled due to a server error.
  • 503 Service Unavailable: Server took too long to fulfill the request.
